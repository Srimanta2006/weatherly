# Modern Weather Dashboard

A clean and minimal weather web application built using HTML, CSS, and JavaScript with real-time weather data from WeatherAPI.

---

## Features

- Real-time weather updates
- 5-day weather forecast
- Dark / Light theme toggle
- Responsive modern UI
- Dynamic weather icons
- Search weather by city
- UV Index display
- Humidity, wind speed, and pressure data
- Minimal and clean dashboard design

---

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- WeatherAPI

---

## API Used

Weather data powered by:

## Installation

Clone the repository:

```bash
https://github.com/Srimanta2006/weatherly/
```

Open the project folder:

```bash
cd modern-weather-dashboard
```

Run the project using Live Server or open `index.html`.

---

## Setup API Key

Replace your API key inside `script.js`:

```js
const apiKey = "YOUR_API_KEY";
```

Get your API key from:

https://www.weatherapi.com/
